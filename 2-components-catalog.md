# Componentes a construir en Storybook

## Atoms
- Button (primary, ghost, subtle, danger · sm, md)
- Icon (SVG sprite, 14/15/16px)
- Badge (neutral, good, warn, danger, info)
- Input (text, numeric, with icon, with suffix)
- Select
- Checkbox
- SegmentedButton (2-5 options)
- Avatar (initials, 22/28/36px)
- Chip (removable)
- Sparkline / Bar mini-chart
- ProgressBar
- Spinner / Empty state

## Molecules
- KpiCard (label, value, delta, optional sparkline)
- StatusBadge (dot + label)
- NavItem (icon + label + badge)
- TableCell (mono, standard, with avatar+subtext)
- FormField (label + input + hint + error)
- BreadcrumbTrail
- FilterPill
- StepperItem

## Organisms
- Sidebar (brand + nav sections + footer)
- TopNav (breadcrumb + search + actions + avatar)
- DataTable (sortable, selectable, with footer totals, sticky header)
- Toolbar (tabs + filters + search)
- AgendaItem (time + duration + type chip + title)
- TaskItem (checkbox + label + due)
- MessageItem (avatar + from + preview + unread dot)
- KanbanStageBar
- FormSection (2-col: description + fields grid)
- StepNav (vertical)
- RightDrawer (header + body + footer)
- Modal (destructive, confirmation, form)

## Templates
- Dashboard grid
- List page (KPIs + toolbar + table)
- Detail page (header + tabs + sections)
- Detail + right panel
- Full form (stepper + form + sticky footer)
- Modal overlay
